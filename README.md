# 📱 INSTALAR NO CELULAR

## Android

1. Abra o app no Chrome
2. Clique no menu ⋮
3. Selecione “Adicionar à tela inicial”
4. Instale o aplicativo

---

## iPhone

1. Abra no Safari
2. Compartilhar
3. “Adicionar à Tela de Início”
